# Call Center Dashboard (Power BI)

## Project Overview
This project is a Call Center Analytics Dashboard built using Power BI.

## Features
- Total Calls Tracking
- Call Duration Analysis
- Response Time Monitoring
- Sentiment Analysis

## Tools Used
- Power BI
- CSV Dataset
